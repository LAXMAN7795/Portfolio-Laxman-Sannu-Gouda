# Laxman Sannu Gouda | Personal Portfolio

A modern, highly interactive, and responsive personal portfolio website showcasing my skills, experience, and projects in Software Engineering, Artificial Intelligence, and Data Science.

## 🌟 Features

- **Striking Neon Aesthetics**: High-contrast dark mode design with vibrant neon cyan and purple accents.
- **Interactive UI/UX**: Features a custom global mouse-tracking glow effect, glassmorphism cards, and typing animations.
- **Fully Responsive**: Seamlessly adapts to any screen size (desktop, tablet, and mobile).
- **Custom Modals & Navigation**: Includes a dynamic "Say Hello" contact modal and smooth-scrolling anchor links.
- **Zero Dependencies**: Built entirely from scratch using pure HTML, CSS, and Vanilla JavaScript for maximum performance.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure and accessibility.
- **CSS3 (Vanilla)**: Flexbox/Grid layouts, custom variables, keyframe animations, and media queries.
- **JavaScript (ES6+)**: DOM manipulation, custom event listeners, and typing effects.
- **FontAwesome**: Scalable vector icons.
- **Google Fonts**: Inter & JetBrains Mono typography.

## 🚀 Projects Showcased

3. **Social SaaS Platform**: AI-powered social media management and scheduling.

## 🏃‍♂️ Running Locally

To view the portfolio locally on your machine, simply run a local web server in the root directory. 

Using Python:
```bash
python -m http.server 8000
```
Then open `http://localhost:8000` in your web browser.
